# POKEDEX
Le Pokédex est un outil essentiel dans l'univers de Pokémon, servant de répertoire des Pokémon. Il permet aux dresseurs de recueillir des informations sur les créatures fictives, telles que leur taille, leur poids, leur type, et leur habitat. Le Pokédex est remis au début de l'aventure par le professeur Pokémon et évolue au fil des générations, ajoutant des fonctionnalités comme la recherche et l'identification des Pokémon. Il existe deux types de Pokédex : les Pokédex régionaux et nationaux, qui recensent les Pokémon selon leur numéro ou leur type.
 ## Présentation
Ce site a pour objectif de :
- Recenser tous les Pokémon connus
- Aider le dresseur à identifier les Pokémon
- Encourager l’exploration et la collection
Le site est destiné à un usage informatif.

## Les technologies utilisées

- TKINTER
